# Climão-IoT

Aplicação ubíqua para ligar tomadas inteligentes com base na temperatura local.

> Trabalho final da disciplina de Computação Móvel e Ubíquoa

**Como rodar**

```shell
cd ClimaSync
npm install
npm start
```

- Precisa configurar as envs
